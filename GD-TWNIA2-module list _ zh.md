# module list

此指令用於列出所有的module資訊。範例如下：

<div style="background-color:black;color:white;padding:20px;">

[user_account@ln01-twnia2 ~]$ module list
Currently Loaded Modules:
  1) intel/18.0.5.274	2) impi/2018.5.274

</div>
