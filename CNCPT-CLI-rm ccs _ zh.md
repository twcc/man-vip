# twccli rm ccs

**`twccli rm ccs`** 指令可刪除 TWCC 容器。

:::warning
{%hackmd @docsharedstyle/tip-zh %}
可使用 **`twccli rm ccs --help`** 列出參數詳細說明
![](https://cos.twcc.ai/SYS-MANUAL/uploads/upload_71f943929ca40c8815599de9b6b62c4d.png =80%x)
:::

## 必要參數

N/A

## 選擇性參數


**`-f, --force`** 
> 強制刪除容器 **(預設：不強制刪除)**

**`-s, --site-id`** (TEXT)
> 指定刪除的容器 ID

**`-h, --help`**
> 取得更多詳細說明
