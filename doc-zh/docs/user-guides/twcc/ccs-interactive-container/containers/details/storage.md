# 容器儲存

TWCC 容器檔案系統已預設為用戶串接 TWCC 高速檔案系統 (HFS)，為專為 AIHPC 所設計的全託管的檔案系統，能夠加速您的運算效能。

您也可使用 TWCC 雲端物件儲存 (COS) 存放容器資料，可透過網路存放和存取各種資料類型（支援S3協定），運用 API 即可完成儲存任務，具備海量空間以及便利介面。

請點選 [高速檔案系統 (HFS)](xxx)、[雲端物件儲存 (COS)](xxx) 以了解更多服務概觀與使用指南。
