sinfo 資訊包含 Slurm 管理的分區和節點的狀態，sinfo 具有多種的過濾、排序，以及格式選項。也很常用於測試 slurm 指令是否可以被正常使用，例如可透過「sinfo -V」來確認Slurm版本資訊。


```
sinfo -V
```
