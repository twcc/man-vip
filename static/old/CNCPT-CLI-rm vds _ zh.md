# twccli rm vds

**`twccli rm vds`** 指令可刪除 TWCC 虛擬磁碟。

:::warning
{%hackmd @docsharedstyle/tip-zh %}
可使用 **`twccli rm vds --help`** 列出參數詳細說明
![](https://cos.twcc.ai/SYS-MANUAL/uploads/upload_c2e06f90ee80b7075644c08acead1bcf.png)


:::

## 必要參數

N/A

## 選擇性參數


**`-f, --force`** 
> 強制刪除資源 (預設：不強制刪除)

**`-id, --disk-id`** (TEXT)
> 指定欲刪除的虛擬磁碟 ID

**`-h`, `--help`**
> 取得詳細說明
