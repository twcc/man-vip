# Merlin Inference

TWCC提供 NGC Merlin Inference 立即可用的工作環境。NVIDIA Merlin 是一個用於在 GPU 上加速整個推薦系統流程的框架：從資料擷取、訓練到部署，使數據科學家、機器學習工程師和研究人員能夠大規模構建高性能推薦系統。透過其包含的工具，可解決常見的 ETL、訓練和推理問題，降低構建深度學習推薦系統的技術門檻。
