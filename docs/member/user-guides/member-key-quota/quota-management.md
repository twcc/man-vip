# 資源配額管理

資源配額管理功能讓租戶管理員可以管理計畫成員使用 TWCC 服務的資源額度，例如容器服務配額或虛擬機器服務。

:::info
注意：此功能需租戶管理員才可使用。
:::

* 點擊右上角使用者名稱，再點選「資源配額管理」。
