# twccli rm vlb

**`twccli rm vlb`** 指令可刪除 TWCC 虛擬磁碟。

:::warning
{%hackmd @docsharedstyle/tip-zh %}
可使用 **`twccli rm vlb --help`** 列出參數詳細說明
![](https://cos.twcc.ai/SYS-MANUAL/uploads/upload_34afbc0ea757907fa33ceeabcedddc31.png)

:::

## 必要參數

N/A

## 選擇性參數


**`-f, --force`** 
> 強制刪除資源 (預設：不強制刪除)

**`-id, --vlb-id`** (TEXT)
> 指定欲刪除的虛擬磁碟 ID

**`-h`, `--help`**
> 取得詳細說明
